  • Q1 What is the difference between - map(), filter(), forEach(), find() ?

    All methods are work with array in Javascript. And each methods are using for different purpose.

    map(): Create a new array and performing on each array element of the original array, and this method return the value.

    filter(): Create a new array by filtering out elements from the original array based on a condition. It returns a new array with only the elements that satisfy the condition. The condition is specified using a callback function, which is applied to each element of the array.

    forEach(): It's same as map() method, create a new array and performing on each array element of the original array, but this method not return any value.

  • Q2 What is the difference between - LocalStorage, sessionStorage, cookies ?

    LocalStorage, sessionStorage, and cookies are all used to store data in web browsers, but they are used for different purposes.

    LocalStorage: This method store the data in the web browser without expiration dates. It is store data in LocalStorage and respond even browser close.

    sessionStorage: This method also store the data in the web browser but the data clear when close the browser or logout the account.

    cookies: this method can be set with an expiration date and can be accessed and modified by the server that created them or by the client-side script running on the web page. Cookies can store small data other than before two methods cookies have 4kb storage and other two store 5mb - 10mb data.

  • Q3 What is event loop? How it's work?

    Event Loop is the mechanism of the Javascript, how Javascript compiler works. this mechanism allow the language to handle multiple concurrent operations in a non-blocking, asynchronous manner. This enables JavaScript to handle multiple tasks concurrently without blocking the main thread and allows for more efficient and scalable web applications.
